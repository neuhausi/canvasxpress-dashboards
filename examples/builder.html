No-code dashboard builder

Click + Data to add an inline data source, then + Panel to add charts. Drag a panel’s title to move it and drag the blue corner to resize (snaps to the 12-column grid). Select a panel to edit its title, data source, chart type, color-by, or raw config. Preview renders it live; Export downloads the spec; Import loads one. Save/Share use the Phase-3 server if one is running on this origin.
